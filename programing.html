CNSS (Certified Network Security Specialist)

By International Cyber Security Institute (ICSI)

Certified Network Security Specialist (CNSS) is one of the courses and certifications provided by the International Cybersecurity Institute (ICSI) which is an institute based in the United Kingdom (UK) and has earned accreditation from the Council for Registered Ethical Security Testers (CREST) and National Cyber Security Centre (NCSC). This course provides a holistic view of modern network security including operating system hardening, firewalls, intrusion-detection systems, VPNs, and Encryption. Physical Security, Standards, System Security, and Security Policies are also included. The course consists of 13 modules, the passing percentage is 70.
